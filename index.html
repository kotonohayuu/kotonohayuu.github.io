📚 プログラミングとは？

プログラミングとは、コンピュータに「どんな動きをしてほしいか」を、私たちの言葉（日本語など）ではなく、コンピュータが理解できる「特別な言葉（プログラミング言語）」で伝えることです。

簡単な例

人間：「おはよう」と言って

コンピュータ：「おはよう」と表示する

たとえば、

  • 人間同士なら「電気をつけて」と言えば伝わりますが、
  • コンピュータには「どうやって電気をつけるか」を細かく順番に命令する必要があります。

もっと身近な例

レシピ（料理の手順）を書くのと似ています。

  1. 材料を用意する
  2. フライパンを熱する
  3. 卵を割り入れる
  4. かき混ぜる
  5. 盛り付ける

このように、やることを一つずつ順番に書くのがプログラミングです。

プログラミングでできること

  • Webサイトやアプリを作る
  • ゲームを作る
  • ロボットを動かす
  • データを自動で整理する など

このサイトで学ぶこと

このサイトでは、Webサイトを作るための3つの言語を学びます：

  • HTML - ページの「骨組み」を作る言語
  • CSS - ページの「見た目・デザイン」を整える言語
  • JavaScript - ページに「動き」や「反応」をつける言語

🏗️ HTML基礎

HTMLは、Webページの「骨組み」や「構造」を作る言語です。家を建てるときの「設計図」や「柱や壁」のような役割をします。

HTMLって何？

HTMLは「HyperText Markup Language（ハイパーテキスト・マークアップ・ランゲージ）」の略です。Webページの内容を、コンピュータが理解できる形で整理して書くための言語です。

身近な例で理解しよう

新聞や雑誌の構成

  • 大きな見出し（タイトル）
  • 中くらいの見出し（章のタイトル）
  • 本文（文章）
  • 写真や図
  • リンク（関連記事への案内）

HTMLも同じように、Webページの各部分を「見出し」「文章」「画像」「リンク」などに分けて整理します。

基本的なHTMLの構造

<!DOCTYPE html>
<html>
<head>
    <title>私の最初のページ</title>
</head>
<body>
    <h1>こんにちは！</h1>
    <p>これは私の最初のWebページです。</p>
</body>
</html>

【< と >（山かっこ）の意味】
HTMLでは、タグの始まりと終わりを<と>（山かっこ）で囲みます。
例：<h1>見出し</h1> のように使います。

よく使うHTMLタグ

<h1> - 大きな見出し
<h2> - 中くらいの見出し
<p> - 段落（文章）
<div> - 箱（グループ化）
<img> - 画像
<a> - リンク
【HTMLタグの詳しい説明】
  • <h1> ～ </h1>：ページで一番大きな見出しを表します。タイトルや重要な項目に使います。
  • <h2> ～ </h2>：<h1>より少し小さい見出しです。章やセクションのタイトルに使います。
  • <p> ～ </p>：文章や説明など、まとまったテキスト（段落）を表します。
  • <div> ～ </div>：いろいろな要素をまとめてグループ化するための「箱」です。デザインやレイアウト調整によく使います。
  • <img>：画像を表示します。src属性で画像ファイルの場所を指定します（例：<img src="sample.jpg">）。
  • <a> ～ </a>：リンクを作ります。href属性でリンク先のURLを指定します（例：<a href="https://example.com">こちら</a>）。

※HTMLタグには必ず終了タグ（例: </h1>）が必要です。
終了タグは</h1>のように、スラッシュ（/）を付けて書きます。

🎨 CSS基礎

CSSは、HTMLで作ったページの「見た目」や「デザイン」を整える言語です。家の「内装」や「外装の色」を決めるような役割をします。

CSSって何？

CSSは「Cascading Style Sheets（カスケーディング・スタイル・シーツ）」の略です。HTMLで作った骨組みに、色や大きさ、配置などの「スタイル」を加えるための言語です。

身近な例で理解しよう

服のデザイン

  • 色（赤、青、緑など）
  • サイズ（S、M、Lなど）
  • 素材（綿、シルク、ウールなど）
  • デザイン（シンプル、派手、カジュアルなど）

CSSも同じように、HTMLの要素に「色」「大きさ」「フォント」「配置」などのスタイルを指定します。

CSSの基本

h1 {
    color: blue;
    font-size: 24px;
    text-align: center;
}

p {
    color: gray;
    font-size: 16px;
    text-align: left;
}

【{ と }（波かっこ）の意味】
CSSでは、どこからどこまでがスタイルの内容かを{と}（波かっこ）で囲みます。
例：h1 { color: blue; } のように使います。

よく使うCSSプロパティ

【h1でよく使うCSSプロパティ】
  • color - 文字の色
  • font-size - 文字の大きさ
  • text-align - 文字の位置（左・中央・右揃え）
  • margin - 見出しの上下余白
【pでよく使うCSSプロパティ】
  • color - 文字の色
  • line-height - 行間
  • margin - 段落の余白
  • text-align - 文字の位置（左・中央・右揃え）
【CSSプロパティの詳しい説明】
  • color：文字の色を指定します。例：color: red;（文字を赤色にする）
  • font-size：文字の大きさを指定します。例：font-size: 24px;（文字を24ピクセルにする）
  • background-color：背景色を指定します。例：background-color: yellow;（背景を黄色にする）
  • margin：要素の外側の余白（スペース）を指定します。例：margin: 10px;（外側に10ピクセルの余白）
  • padding：要素の内側の余白（スペース）を指定します。例：padding: 10px;（内側に10ピクセルの余白）
  • border：枠線をつけます。例：border: 1px solid black;（1ピクセルの黒い実線の枠）
  • text-align：文字の位置を指定します。例：text-align: center;（中央揃え）、text-align: right;（右揃え）
【h1とpの役割・使い方】
  • h1：ページで一番大きな見出しです。タイトルや重要な項目に使います。CSSで色や大きさを変えることで、より目立たせることができます。
    • 例：color（文字色）、font-size（文字の大きさ）、text-align（中央揃え）
    • 例：h1 { color: blue; font-size: 32px; text-align: center; }
  • p：文章や説明など、まとまったテキスト（段落）を表します。CSSで文字色や行間、余白などを調整できます。
    • 例：color（文字色）、line-height（行間）、margin（段落の余白）
    • 例：p { color: gray; line-height: 1.8; margin: 16px 0; }

【border（枠線）とは？】
borderは「枠線」をつけるプロパティです。たとえば、文章や画像のまわりに線をつけて目立たせたいときに使います。
例：border: 1px solid black;（1ピクセルの黒い実線の枠をつける）

【:（コロン）の意味】
CSSでは「プロパティ名: 値;」のように、コロン（:）で「何をどうするか」を指定します。
例：color: red;（色を赤にする）

⚡ JavaScript基礎

JavaScriptは、Webページに「動き」や「反応」を加える言語です。家の「電気スイッチ」や「リモコン」のような役割をします。

JavaScriptって何？

JavaScriptは、Webページを「インタラクティブ（双方向的）」にするための言語です。ユーザーの操作（クリック、入力など）に反応して、ページの内容を動的に変更できます。

身近な例で理解しよう

テレビのリモコン

  • ボタンを押すとチャンネルが変わる
  • 音量ボタンで音の大きさを調整
  • 電源ボタンでテレビのオン/オフ
  • メニューボタンで設定画面を表示

JavaScriptも同じように、ボタンをクリックしたり、フォームに入力したりすると、ページの内容が変わったり、新しい情報が表示されたりします。

JavaScriptの基本

// 文字を表示する
console.log("こんにちは！");

// 変数（データを保存する箱）
let name = "太郎";
console.log("こんにちは、" + name + "さん！");

// 関数（処理をまとめたもの）
function sayHello() {
    alert("こんにちは！");
}

よく使うJavaScript

アラートを表示

alert("メッセージ");

文字を変更

document.getElementById("id").innerHTML = "新しい文字";

この文字が変わります

【JavaScriptの基本用語・使い方の解説】
  • console.log()：コンソール（開発者ツール）にメッセージや値を表示します。プログラムの動きを確認したいときに便利です。
  • let：変数（データを入れる箱）を作るときに使います。例：let name = "太郎";
  • function：処理をまとめて「関数」として定義します。例：function sayHello() { ... }
  • alert()：画面にポップアップでメッセージを表示します。例：alert("こんにちは！");
  • document.getElementById()：HTMLの特定の要素（idで指定）をJavaScriptから操作できます。
  • innerHTML：HTML要素の中身（テキストやタグ）を書き換えます。
